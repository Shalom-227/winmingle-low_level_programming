# 0x01. C - Variables, if, else, while

This project introduces fundamental programming concepts in C, including:

- Variables and data types
- Conditional statements (if, else)
- Loops (while)
- Basic number manipulation
- Output formatting using putchar and printf

## Files and Tasks

### Programs  

+ 0-positive_or_negative.c
  Generated a random number and determine whether number is positive, negative, or zero.

+ 1-last_digit.c
  Prints the last digit of a random number with conditions.

+ 2-print_alphabet.c
  Prints lowercase alphabet using only putchar.

+ 3-print_alphabets.c
  Prints lowercase and uppercase alphabets.

+ 4-print_alphabt.c
  Prints lowercase alphabet except q and e.

+ 5-print_numbers.c
  Prints digits from 0 to 9.

+ 6-print_numberz.c
  Prints digits using only putchar (no char).

+ 7-print_tebahpla.c
  Prints alphabet in reverse.

+ 8-print_base16.c
  Prints hexadecimal numbers.

+ 9-print_comb.c
  Prints single-digit combinations separated by commas.

+ 100-print_comb3.c
  Prints all unique combinations of two digits numbers.

+ 101-print_comb4.c
  Prints all unique combinations of three digits numbers.

+ 102-print_comb5.c
  Prints all combinations of two two-digit numbers.

## Usage

+ Set the C file to be used:

  export CFILE=main.c

+ Run scripts:
  ./0-positive_or_negative.c
  ./1-last_digit.c
  ./2-print_alphabet.c
  ./3-print_alphabets.c
  ./4-print_alphabt.c
  ./5-print_numbers.c
  ./6-print_numberz.c
  ./7-print_tebahpla.c
  ./8-print_base16.c
  ./9-print_comb.c
  ./100-print_comb3.c
  ./101-print_comb4.c
  ./102-print_comb5.c

+ Compile C programs:
  
  gcc -Wall -Werror -Wextra -pedantic file.c -o output

+ ./output


## Requirements

- Allowed editors: vi, vim, emacs
- Compiled on Ubuntu using gcc -Wall -Werror -Wextra -pedantic -std=gnu89
- No use of printf unless explicitly allowed
- Use of putchar is required for most tasks
- Code must follow Betty coding style
- All programs must end with a new line
